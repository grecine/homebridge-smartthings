# Homebridge SmartThings (OAuth) — Wiki

A modern Homebridge plugin that connects your SmartThings devices to Apple HomeKit. This wiki contains the full documentation; the [README](https://github.com/aziz66/homebridge-smartthings#readme) has the quick-start.

## Get started

- **[Installation and OAuth Setup](https://github.com/aziz66/homebridge-smartthings/wiki/Installation-and-OAuth-Setup)** — install the plugin, create a SmartThings OAuth app, and run the wizard.
- **[Supported Devices and Capabilities](https://github.com/aziz66/homebridge-smartthings/wiki/Supported-Devices-and-Capabilities)** — every SmartThings capability the plugin recognizes and the HomeKit service it maps to.

## Optional features

- **[Webhooks and Real-Time Updates](https://github.com/aziz66/homebridge-smartthings/wiki/Webhooks-and-Real-Time-Updates)** — push events from SmartThings instead of polling, for instant device updates.
- **[Samsung Frame TV](https://github.com/aziz66/homebridge-smartthings/wiki/Samsung-Frame-TV)** — true power off, Art Mode switch, and TV app launcher for Samsung TVs.
