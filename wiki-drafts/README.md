Source markdown for the GitHub Wiki at <https://github.com/aziz66/homebridge-smartthings/wiki>.

## Publishing

1. Make sure Wikis are enabled for the repo: **Settings → Features → Wikis**.
2. In the GitHub UI, go to the repo's **Wiki** tab.
3. For each file above, create a new wiki page using the title in the right column and paste the file contents.
4. Save. Internal links in the README and between wiki pages will resolve immediately.
